# v1.2 web-console

这是“宅宅能量条系统”的 v1.2 网页控制台与单一后端状态源快照。

## 本版重点

- 新增 FastAPI 网页控制台入口：`GET /` 返回原生 HTML/CSS/JavaScript 页面。
- 网页端通过 `GET /state` 读取统一 `GameState`，通过 `POST /command` 执行所有写操作。
- 网页端接入行动番茄钟流程：选择时长、本地倒计时、暂停、继续、放弃、完成后发送 `COMPLETE_TIMED_ACTION`。
- 网页端补齐 Tkinter 主要能力：每日任务、特殊任务、刷新任务、商店、成就、称号查看与称号装备。
- 商店、成就、称号在网页端改为二级弹窗，不再全部堆在主界面。
- Tkinter 改为 HTTP 客户端，和网页端共用 FastAPI 中的同一个 `GameCore()` 状态源。
- 行动经验奖励改为由 `actions.json` 的 `exp_change` 数据驱动，不再由 Python 按行动名称或正负硬编码。
- 数据文件路径统一解析到项目根目录，并支持测试通过 `OTAKU_ENERGY_DATA_DIR` 覆盖。
- 扩展 API、客户端、架构、路径、计时行动和网页静态资源测试。

## 运行方式

```bash
uvicorn api:app --reload
```

访问：

- `http://127.0.0.1:8000/`
- `http://127.0.0.1:8000/docs`
- `http://127.0.0.1:8000/state`

Tkinter 本地端：

```bash
python main.py
```

运行 Tkinter 前需要先启动 FastAPI 后端。
