# v1.0 achievement-title-timer-ui

这是“宅宅能量条系统”的 v1.0 历史快照。

## 本版重点

- 成就系统扩充为正式配置，支持经验和金币奖励。
- 成就条件新增特殊任务完成数、等级、任务组合等类型。
- 行动系统加入计时机制，完成倒计时后统一结算。
- 正向行动按 25/45/60 分钟倍率增加能量。
- 娱乐行动按 30/60/90 分钟消耗能量，并获得少量经验。
- 称号系统扩充到 13 个称号，并支持更多解锁条件和加成效果。
- 主界面重排为标题、状态概览、导航、行动与任务、最近行动记录等区域。
- 新增 `themes.py`，集中管理默认主题、字体、间距和尺寸。
- 补充成就、计时行动、称号、主题等测试。

## 快照包含

- Python 源码：`main.py`、`ui.py`、`player.py`、`task.py`、`achievement.py`、`title.py`、`level.py`、`shop.py`、`themes.py`
